# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-29

### Added
- **E-Commerce Features**
  - Complete product catalog with categories
  - Shopping cart with quantity management
  - Checkout flow with address and payment selection
  - Order history with detailed status tracking

- **Training Programs**
  - Professional training programs by expert trainers
  - Advanced filtering by difficulty, duration, and goals
  - Detailed program views with workout breakdowns

- **AI Assistant**
  - Chat-based AI for supplement recommendations
  - Persistent chat history for logged-in users
  - Quick question suggestions
  - Context-aware responses

- **User Features**
  - Authentication (Login/Register/Guest mode)
  - Profile management
  - Address book management
  - Payment methods management
  - Favorites/Wishlist
  - Notification preferences

- **Customization**
  - Dark/Light theme with system preference detection
  - Multi-language support (English, Georgian)
  - Responsive design for all screen sizes

- **UI/UX**
  - Premium glassmorphism design
  - Smooth animations and transitions
  - Bottom sheet modals
  - Pull-to-refresh on all screens
  - Toast notifications

### Technical
- React Native 0.81.5 with Expo SDK 54
- TypeScript for type safety
- React Navigation for routing
- AsyncStorage for data persistence
- Platform-specific optimizations (iOS, Android, Web)
